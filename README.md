# Tic Tac Toe Game in C
I created this Tic Tac Toe game using the C programming language. 
It is a console-based application where two players can play the game by taking turns.
The game is played on a 3Ã—3 board. Players enter their choice through the keyboard.
The program checks for win conditions and also handles draw situations.

## Features
- Two player game
- Simple console interface
- Win and draw checking
- Valid move checking

## Concepts Used
- C programming
- Arrays
- Functions
- Loops
- If-else conditions

## How to Run
1. Compile the program using:
   gcc tictactoe.c -o tictactoe
2. Run the program:
   ./tictactoe
